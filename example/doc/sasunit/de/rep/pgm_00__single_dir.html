new_description
Ausführliche Beschreibung
Generates a dataset with the names of all files in a directory.
Wildcards may be used to specify the files to be included
 
Resulting SAS dataset has the columns
membername (name of the file in the directory)
filename (name of file with absolute path, path separator is slash)
changed (last modification data as SAS datetime).

Parameter
i_dsPath name of sas dataset containing one variable directory. For each observation/directory the search will be done.
i_pattern pattern which the filemembers must match
o_members output dataset, default is work.dir. This dataset contains three columns
  named membername, filename and changed (optional: default = work.dir)
o_subDirs output dataset. This dataset contains one column named directory and
  holds all subdirectories (optional: default = _NONE_)
