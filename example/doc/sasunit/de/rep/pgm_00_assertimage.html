Ausführliche Beschreibung
This assert compares images on equality. It's an implementation to show how to use assertExternal for a specific purpose.
 
As compare tool imageMagick is used. Make sure to use version 6.9 or above since before the compare method always returned 0.
Beginning with ImageMagick 6.9 the following return codes are used:
0: images match
1: images different
 
Please refer to <A href="https://sourceforge.net/p/sasunit/wiki/User's%20Guide/" target="_blank">SASUnit User's Guide</A>

Parameter
i_script Path of shell script
i_expected Path of first image file (expected)
i_actual Path of second image file (actual)
i_expected_shell_rc Optional parameter: Expected return value of called script i_script (default = 0)

i_modifier Optional parameter: modifiers for the compare (default = "-metric RMSE")
i_threshold Optional parameter: further parameter to be passed to the script. Default is 0. To be used especially with

i_desc Optional parameter: description of the assertion to be checked (default = "Comparison of images")
