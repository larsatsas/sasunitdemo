Program Documentation for Program _reportpgmheader.sas

new_description
Detailed description
Render header section of program documentation
 
Works on a pre-defined dataset and separates simple tags from complex ones.
 
Simple tags are those that consists of a tagname and a value like \\brief or \\author
Complex tags consist of tagname parameter name and a value like \\retval or \\param
